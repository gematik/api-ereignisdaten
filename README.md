# API-Ereignisdaten

The repository contains a description of the endpoints for data delivery according to the gemSpec_perf specification – see section “2.5.7 Ereignisdaten” ([link](https://gemspec.gematik.de/docs/gemSpec/gemSpec_Perf/latest/#2.5.7)).
The following pages describe the parameters and data types of the web service for the endpoints.

The [doucmentation](content/Documentation.md) is available in **German only**.

Die **ausschließlich auf Deutsch** zur Verfügung stehende **[Dokumentation](content/Documentation.md)** findet sich im Content folder.
